# Gmod-BOT
A simple counter bot made in discord.js (and gamedig) for gmod.

## Installation:
***
A little intro about the installation. 
```
$ git clone https://github.com/ach-git/Gmod-BOT
$ cd ../Gmod-BOT
$ npm install discord.js
// Edit the config.json file
$ node index.js
```
